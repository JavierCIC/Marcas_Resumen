    1 Variables y Tipos

    Las variables son como cajas donde guardamos información. Usamos let cuando el valor puede cambiar, const cuando no va a cambiar nunca, y var (mejor evitarla). Los tipos más comunes son texto (string), números, verdadero/falso (boolean) y objetos.

    let nombre = "Laura";
const EDAD = 20;
let esAlumno = true;

console.log(typeof nombre); // "string"

    Explicación del ejemplo: Aquí declaramos tres variables de forma diferente. let se puede cambiar después, const no se puede modificar, y mostramos el tipo de dato con typeof.

    2 Operadores

    Los operadores nos permiten hacer operaciones con los datos. Tenemos operadores matemáticos (+, -, *, /, %), de comparación (>, <,===) y lógicos (&&, ||) que nos ayudan a tomar decisiones.

    let a = 15;
let b = 4;
console.log(a + b);     // 19
console.log(a > b);     // true
console.log(a % b);     // 3

    Explicación del ejemplo: Usamos operadores matemáticos (+ y %) y de comparación (>). El resultado de cada operación se muestra en la consola.

    3 Condicionales

    Los condicionales nos permiten ejecutar código solo cuando se cumple una condición. Es como decir "si pasa esto, haz esto otro". Se usan mucho con if, else if y else.

    let nota = 7;
if (nota >= 9) {
    console.log("Sobresaliente");
} else if (nota >= 5) {
    console.log("Aprobado");
} else {
    console.log("Suspenso");
}

    Prueba tu nota:

    Explicación del ejemplo: Según el valor de la variable nota, el programa decide qué mensaje mostrar. Es la base de la lógica en programación.

    4 Bucles

    Los bucles sirven para repetir código varias veces sin tener que escribirlo manualmente. El más usado es el for, pero también están while y for...of para recorrer arrays, que son los próximos.

    for(let i = 0; i < 5; i++) {
    console.log("Número:", i);
}

    Explicación del ejemplo: Este bucle for repite el código 5 veces, mostrando los números del 0 al 4. Muy útil para repetir tareas.

    5 Arrays

    Un array es una lista de elementos. Puedes guardar varios valores en una sola variable. Tiene métodos útiles como push() para añadir, pop() para quitar, y map(), filter() para trabajar con ellos.

    let frutas = ["manzana", "plátano"];
frutas.push("naranja");
console.log(frutas.length);
console.log(frutas[0]);

    Explicación del ejemplo: Creamos una lista de frutas, añadimos una nueva con push() y mostramos cuántas hay y la primera fruta.

    6 Funciones

    Las funciones son bloques de código que podemos reutilizar. Sirven para organizar mejor el programa y evitar repetir código. Pueden recibir datos (parámetros) y devolver un resultado.

    function sumar(a, b) {
    return a + b;
}

const multiplicar = (a, b) => a * b;

    Explicación del ejemplo: Creamos dos funciones para sumar y multiplicar. La arrow function (`=>`) es una forma más moderna y corta de escribir funciones.

    7 DOM

    El DOM es la forma en que JavaScript puede acceder y modificar la página web. Con él podemos cambiar textos, colores, añadir o quitar elementos, todo desde nuestro código.

    document.getElementById("titulo").innerText = "Nuevo texto";
document.querySelector(".parrafo").style.color = "yellow";

    Explicación del ejemplo: Estas líneas cambian el texto y el color de elementos de la página web usando JavaScript.

    8 Eventos

    Los eventos son cosas que pasan en la página (hacer click, pulsar una tecla, mover el ratón...). Con addEventListener podemos hacer que nuestro código reaccione cuando ocurra algo.

    boton.addEventListener("click", () => {
    console.log("Botón pulsado");
});

    Explicación del ejemplo: Con addEventListener podemos hacer que el código reaccione cuando el usuario haga click en un botón.

    9 Objeto Math

    El objeto Math nos da funciones matemáticas útiles como redondear números, generar números aleatorios, sacar raíces cuadradas, etc. Es muy útil para hacer cálculos.

    Explicación del ejemplo: Usamos Math.random() y Math.floor() para generar un número aleatorio entre 1 y 100.

    10 Strings

    Los strings son textos. JavaScript tiene muchas funciones útiles para trabajar con ellos: pasarlos a mayúsculas, minúsculas, saber su longitud, unirlos, etc.

    let texto = "Hola mundo";
console.log(texto.toUpperCase());
console.log(`Me llamo ${nombre}`);

    Explicación del ejemplo: Convertimos el texto a mayúsculas y usamos template literals (las comillas invertidas ``) para combinar texto con variables.

    11 Fechas (Date)

    El objeto Date nos permite trabajar con fechas y horas. Podemos crear una fecha, obtener el día, mes, año, y mostrarla de diferentes formas.

    let hoy = new Date();
console.log(hoy.getFullYear());
console.log(hoy.toLocaleDateString());

    Explicación del ejemplo: Creamos un objeto fecha con la fecha actual y mostramos el año y la fecha completa en formato local.

    12 Formularios

    Los formularios nos permiten recoger información del usuario. Con JavaScript podemos leer lo que escribe, validarlo y decidir qué hacer con esa información.

    Explicación del ejemplo: Al enviar el formulario, JavaScript lee el nombre escrito y puede validarlo antes de enviarlo.
